Do you love to play checkers? Why don't you install it and let's play together

## Getting Started

To start run

```bash
npm install
#or
yarn install
```

And, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
